Chapter 5

Helpful Resources

Primary documentation for the runtime, the ORM, the DI container, and the design ideas behind the pattern.

Platform

Runtime and Language

The SDK, the language, and the project format the guide uses.

.NET Documentation

The hub for the SDK, the runtime, and the command-line tooling used to build and run the project.

https://learn.microsoft.com/dotnet/

C# Language Reference

Covers primary constructors, collection expressions, and nullable reference types, all of which appear in the example code.

https://learn.microsoft.com/dotnet/csharp/
Libraries

ORM and Dependency Injection

The two packages that make the pattern concrete.

Entity Framework Core

The ORM behind the Database Context and the in-memory provider used throughout the guide. Read the DbContext and provider pages first.

https://learn.microsoft.com/ef/core/

Dependency Injection in .NET

Explains the service collection, service lifetimes (scoped, transient, singleton), and how the provider builds the object graph.

https://learn.microsoft.com/dotnet/core/extensions/dependency-injection

System.Text.Json

The serializer the Import Service uses to deserialize the external source, including property naming and case-insensitive binding.

https://learn.microsoft.com/dotnet/standard/serialization/system-text-json/
Design

The Ideas Behind the Pattern

Background reading on the principles the layers express.

Dependency Inversion Principle

The principle that high-level layers should depend on abstractions, not concrete implementations, which is exactly why each seam is an interface.

https://en.wikipedia.org/wiki/Dependency_inversion_principle

Repository Pattern

Background on putting a collection-like interface in front of a data store so callers stay ignorant of persistence details.

https://martinfowler.com/eaaCatalog/repository.html

Inversion of Control Containers

A longer essay on the composition root idea: why one place owns construction and the rest of the program receives its dependencies.

https://martinfowler.com/articles/injection.html
